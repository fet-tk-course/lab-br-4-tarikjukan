# Lab 4 — Python za FastAPI

## Priprema

1. Prihvatite assignment link koji vam je poslao asistent
2. Klonirajte vaš repozitorij:
```
   git clone https://github.com/fet-tk-course/lab-4-githubusername
```
3. Otvorite folder u VS Code
4. Kreirajte novi fajl: `lab04_vašeime.py`

## Struktura repozitorija

Na kraju vježbe vaš repozitorij treba sadržavati:
```
lab05-python-fastapi/
├── lab04_vašeime.py     ← vaš kod
└── prompts.md           ← promptovi koje ste koristili za Copilot
```

## Predaja

Kada završite, uradite push kod:
```
git add .
git commit -m "Sedmica 4 — Python za FastAPI"
git push
```

Provjerite na GitHubu da su fajlovi vidljivi prije nego što odete s vježbe.

- 🤖 **Koristite Copilot** — AI je dozvoljen. Promptove koje koristite zapišite u `prompts.md`
